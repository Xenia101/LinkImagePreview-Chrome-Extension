이미지 URL 미리보기 테스트 페이지

이 페이지는 이미지 URL 미리보기 확장 프로그램을 테스트하기 위한 페이지입니다. 아래의 다양한 형식의 이미지 URL에 마우스를 올려보세요.

참고: 미리보기가 표시되지 않는다면, 확장 프로그램이 제대로 설치되었는지 확인하세요.

1. 간단한 이미지 URL

여기 샘플 이미지 URL이 있습니다: https://placehold.co/200x300.jpg

2. 다양한 이미지 형식

  • PNG 이미지: https://placehold.co/200x300.png
  • GIF 이미지: https://placehold.co/200x300.gif
  • JPEG 이미지: https://placehold.co/200x300.jpeg

3. 문장 내 이미지 URL

이 문장에는 https://placehold.co/200x300.jpg?text=Sample+Image 이미지 URL이 포함되어 있습니다. 확장 프로그램이 이를 감지하여 미리보기를 표시해야 합니다.

4. 쿼리 파라미터가 있는 URL

https://placehold.co/200x300.jpg?text=HelloWorld&font=roboto 이와 같이 쿼리 파라미터가 있는 URL도 지원합니다.

5. 컬러 지정 이미지

https://placehold.co/200x300/FF0000/FFFFFF.jpg 이와 같이 빨간 배경에 흰색 텍스트로 된 이미지도 지원합니다.

6. Picsum 랜덤 이미지

https://picsum.photos/200/300.jpg 피쿰의 랜덤 이미지 URL도 지원합니다.
