Image URL Preview Test Page

This page is designed to test the image URL preview extension. Hover your mouse over the various image URLs below.

Note: If previews don't appear, check if the extension is properly installed.

1. Simple Image URL

Here's a sample image URL: https://placehold.co/200x300.jpg

2. Various Image Formats

  • PNG image: https://placehold.co/200x300.png
  • GIF image: https://placehold.co/200x300.gif
  • JPEG image: https://placehold.co/200x300.jpeg

3. Image URL Within Text

This sentence contains an image URL: https://placehold.co/200x300.jpg?text=Sample+Image. The extension should detect this and display a preview.

4. URL With Query Parameters

https://placehold.co/200x300.jpg?text=HelloWorld&font=roboto URLs with query parameters like this are also supported.

5. Color-Specified Image

https://placehold.co/200x300/FF0000/FFFFFF.jpg Images with red background and white text like this are also supported.

6. Picsum Random Images

https://picsum.photos/200/300.jpg Picsum's random image URLs are also supported.

7. Dynamically Added Content
